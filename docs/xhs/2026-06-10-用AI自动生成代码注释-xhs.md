### AI自动生成代码注释的步骤

1. **数据收集**：收集大量的代码注释数据，包括源代码和注释。
2. **模型训练**：用机器学习算法训练模型，让它学会生成注释。
3. **代码分析**：分析代码的结构和语义，为生成注释做准备。
4. **注释生成**：模型根据分析结果自动生成注释。

### 代码示例

来看看一个简单的Java代码示例，感受一下AI的魔力：

```java
import java.util.ArrayList;
import java.util.List;

public class CodeCommentGenerator {

    private List<String> generateComments(String code) {
        List<String> comments = new ArrayList<>();
        // 代码分析
        String[] lines = code.split("\\n");
        for (String line : lines) {
            if (line.contains("public") || line.contains("private") || line.contains("protected")) {
                comments.add("This is a public/private/protected method/class/variable.");
            } else {
                comments.add("This is a regular line of code.");
            }
        }
        return comments;
    }

    public static void main(String[] args) {
        CodeCommentGenerator generator = new CodeCommentGenerator();
        String code = "public class Example {\n" +
                      "    public void printHello() {\n" +
                      "        System.out.println(\"Hello, world!\");\n" +
                      "    }\n" +
                      "    private int number;\n" +
                      "}";
        List<String> comments = generator.generateComments(code);
        for (String comment : comments) {
            System.out.println(comment);
        }
    }
}
```
