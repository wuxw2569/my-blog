📝 **为什么需要 AI 帮手**？

想象一下，你刚完成一个酷炫的项目，现在要写 README。😓 你需要：

- 涵盖项目介绍、安装、使用、贡献等信息
- 遵循特定的格式，比如 Markdown
- 确保信息准确无误

🎯 **AI 的神奇之处**

1. **自动生成文档结构**：AI 会根据项目类型和内容，自动生成文档的基本结构。
2. **智能填充内容**：利用自然语言处理技术，AI 可以从项目代码、文档中提取信息，填充到文档中。
3. **格式化输出**：AI 会自动将内容按照 Markdown 格式进行输出。

1. **安装库**：`pip install gitpython`
2. **编写脚本**：使用 Python 编写一个简单的 AI 助手。
3. **运行脚本**：在项目目录下运行脚本，生成 README 文件。
