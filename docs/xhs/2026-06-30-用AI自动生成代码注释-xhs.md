### 如何实现？

1. **代码解析**：AI会像侦探一样，深入你的代码，提取出变量、函数、类等信息。
2. **语义理解**：然后，AI会像魔法师一样，理解代码的意图和功能。
3. **模板生成**：最后，AI会像程序员一样，根据提取的信息，生成注释模板。

📝 举个例子，如果你写了一段Java代码，AI就能自动生成相应的注释，就像这样：

```java
public int sum(int a, int b) {
    // @author ai-helper
    // @Description 计算两个数字的和
    // @param a 第一个数字
    // @param b 第二个数字
    return a + b;
}
```
